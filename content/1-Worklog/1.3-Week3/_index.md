### Week 3 Objectives:
* Deploy an application with automatic scaling and load balancing using Auto Scaling and ELB.
* Track, manage, and optimize AWS spending with AWS Budgets.
* Observe systems, resources, and applications in depth with Amazon CloudWatch.

### Tasks to be carried out this week:

### Week 3 Achievements:
* **Scalable Architecture (Auto Scaling & ELB):** Configured health checks and target groups to ensure only healthy instances receive traffic, and validated automatic scale-out/scale-in behavior under simulated load using target tracking policies.
* **Cost Control (AWS Budgets):** Built granular budgets segmented by service and tag, integrated SNS for real-time alert delivery, and used Cost Explorer to identify historical spending patterns and cost optimization opportunities.
* **System Monitoring (CloudWatch):** Created custom metrics from log data and configured Composite Alarms to reduce alert noise from false positives, while using Logs Insights queries to quickly pinpoint the root cause of application errors.
